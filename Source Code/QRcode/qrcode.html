Wi-Fi QR Code Scanner

Scan the QR code to automatically reveal the Wi-Fi network details

Scanner

How to enable camera access:

  1. Click the camera icon in your browser's address bar
  2. Select "Always allow" for camera access
  3. Refresh this page

Network Details

Scan a Wi-Fi QR code to see the network details here.

Wi-Fi QR codes typically look like this and contain the network name and password:

WIFI:S:MyNetwork;T:WPA;P:MyPassword;;

How to use this scanner

  1. Click "Start Scanner" and allow camera access when prompted
  2. Hold the QR code steady in front of your camera
  3. The network details will appear automatically
  4. Alternatively, upload an image containing the QR code
